# Phone Customization Studio 📱
[English](README.md) | [Русский](README.ru.md)

🔗 **[Live Demo](https://davidsulava.github.io/phone-customisation-studio/)**

Интерактивное веб-приложение для кастомизации чехлов смартфонов, построенное на стеке **Next.js 15** и **React 19**. Сервис позволяет пользователям создавать уникальный дизайн чехла в реальном времени.

### ✨ Основные возможности
*   **Визуальный редактор**: Настройка дизайна с использованием `react-rnd` для свободного перемещения и изменения размеров элементов.
*   **Инструменты кастомизации**: Выбор цветов (через `react-color`), работа с изображениями и слоями.
*   **Экспорт результата**: Генерация готовых макетов в формате изображений (`html-to-image`) и возможность сохранения в ZIP-архив.
*   **Современный интерфейс**: Адаптивный UI на базе **Tailwind CSS** и компонентов **Radix UI**, дополненный плавной анимацией от **Framer Motion**.

### 🛠 Технологический стек
*   **Framework**: Next.js 15 (App Router), TypeScript
*   **UI Library**: Radix UI (Primitives), Lucide React (Icons)
*   **Styling**: Tailwind CSS, class-variance-authority
*   **Utilities**: React Hook Form, JSZip, React Dropzone
