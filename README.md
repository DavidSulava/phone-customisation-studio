# Phone Customization Studio 📱
[English](README.md) | [Русский](README.ru.md)

🔗 **[Live Demo](https://davidsulava.github.io/phone-customisation-studio/)**

An interactive web application for customizing smartphone cases, built with **Next.js 15** and **React 19**. This tool allows users to design their own unique phone cases in real-time.

### ✨ Key Features
*   **Visual Editor**: Design customization using `react-rnd` for flexible moving and resizing of elements.
*   **Customization Tools**: Color selection (via `react-color`), image uploads, and layer management.
*   **Export Options**: Generate ready-to-use mockups as images (`html-to-image`) and save designs as ZIP archives.
*   **Modern UI**: Fully responsive interface powered by **Tailwind CSS** and **Radix UI**, featuring smooth animations with **Framer Motion**.

### 🛠 Tech Stack
*   **Framework**: Next.js 15 (App Router), TypeScript
*   **UI Library**: Radix UI (Primitives), Lucide React (Icons)
*   **Styling**: Tailwind CSS, class-variance-authority
*   **Utilities**: React Hook Form, JSZip, React Dropzone
